# CourierConnect Frontend

React frontend for CourierConnect built with Vite and TailwindCSS.

## Installation

```bash
cd client
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Features

- **Home Page**: Landing page with features and CTA
- **Send Parcel**: Form to create a new parcel request
- **Post Trip**: Form for travelers to post their trips
- **Matches**: Find matching trips for parcels
- **Bookings**: Manage bookings between parcels and trips

## Tech Stack

- React 18
- Vite
- React Router DOM
- TailwindCSS
- Axios
